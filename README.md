# DOCUMENTACION JUEGO DE BOLOS

### Ejecución de pruebas
Ejecutar el comando:
```
mvn clean test
```

### Construcción del componente
Ejecutar el comando:
```
mvn clean install
```

### Ejecución de programa
Ejecutar el comando:
```
mvn exec:java -Dexec.mainClass=com.susanibar.david.CampeonatoBolos -Dexec.args=file/puntajeporjugador.txt
```
